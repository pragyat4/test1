# Customer Form Project

A simple, responsive web application that allows users to collect and display customer information in a table format.

## Features

- **Customer Information Form**: Collect customer details including:
  - Name
  - Email
  - Phone
  - Age

- **Dynamic Table Display**: Submitted data is displayed in a organized table format
- **Form Validation**: Ensures all fields are filled before adding to the table
- **Auto-clear Form**: Form automatically resets after successful submission
- **Responsive Design**: Clean, modern UI with a pleasant color scheme

## Technologies Used

- **HTML5**: Semantic markup and form structure
- **CSS3**: Responsive styling with a cyan color theme
- **JavaScript**: Form handling and dynamic table manipulation

## Colors

- Primary Color: Cyan (#00bcd4)
- Hover Color: Dark Cyan (#0097a7)
- Background: Light Gray (#f4f6f9)
- Container: White with shadow effect

## How to Use

1. Open `index.html` in your web browser
2. Fill in the Customer Form with the following information:
   - Enter your name
   - Enter your email address
   - Enter your phone number
   - Enter your age
3. Click the "Add to Table" button
4. Your information will appear in the table below
5. The form will automatically clear for the next entry
